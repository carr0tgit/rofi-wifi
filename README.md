<h3>rofi-wifi</h3>
</div>
<div align="center">Simple script that creates a rofi menu to connect/disconnect networks</div>

## Installation 
1. Install dependencies: iwctl, nmcli, rofi
2. clone repository
3. `cd rofi-wifi`
4. `./rofi-wifi`
5. Add script to `$PATH` for easy access
